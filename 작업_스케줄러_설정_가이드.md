# 티스토리 자동 포스팅 - 작업 스케줄러 설정 가이드

## 📋 개요
이 가이드는 Windows 작업 스케줄러를 사용하여 티스토리 자동 포스팅을 **하루 5번** 자동 실행하도록 설정하는 방법입니다.

## ⏰ 실행 시간표 (미장 실시간 + 브리핑 스타일 합본)

| 시간 | 목적 |
|------|------|
| **23:30 (밤 11시 30분)** | 🔔 미장 개장 속보 |
| **03:00 (새벽 3시)** | 📊 중간 업데이트 |
| **07:00 (아침 7시)** | 📰 마감 정리 + 어젯밤 종합 |
| **12:00 (점심 12시)** | 🍜 점심 브리핑 |
| **19:00 (저녁 7시)** | 🎯 오늘의 전망 |

---

## 🚀 설정 방법

### 1단계: 작업 스케줄러 열기
1. `Win + R` 키를 누르고 `taskschd.msc` 입력 후 엔터
2. 또는 시작 메뉴에서 "작업 스케줄러" 검색

### 2단계: 새 작업 만들기 (5번 반복)
1. 오른쪽 패널에서 **"기본 작업 만들기"** 클릭
2. 이름 입력 (아래 표 참고)
3. **다음** 클릭

### 작업 이름 표

| 순서 | 작업 이름 | 시간 설정 |
|------|----------|----------|
| 1 | `티스토리 자동 포스팅 - 23시30분 (개장속보)` | 오후 11:30:00 |
| 2 | `티스토리 자동 포스팅 - 03시00분 (중간업데이트)` | 오전 3:00:00 |
| 3 | `티스토리 자동 포스팅 - 07시00분 (마감정리)` | 오전 7:00:00 |
| 4 | `티스토리 자동 포스팅 - 12시00분 (점심브리핑)` | 오후 12:00:00 |
| 5 | `티스토리 자동 포스팅 - 19시00분 (오늘전망)` | 오후 7:00:00 |

### 3단계: 트리거 설정
1. **"매일"** 선택 → 다음
2. 시작 시간: **위 표 참고**
3. 간격: **1일마다**
4. **다음** 클릭

### 4단계: 작업 설정
1. **"프로그램 시작"** 선택 → 다음
2. 프로그램/스크립트: 
   ```
   C:\Users\66683\OneDrive\바탕 화면\연구자동화\stock-empire\티스토리_자동_포스팅.bat
   ```
3. 시작 위치 (선택사항):
   ```
   C:\Users\66683\OneDrive\바탕 화면\연구자동화\stock-empire
   ```
4. **다음** 클릭

### 5단계: 최종 확인
1. 설정 내용 확인
2. **"속성 대화 상자 열기" 체크**
3. **마침** 클릭

### 6단계: 고급 설정 (속성 창에서)
1. **"일반"** 탭:
   - ✅ "가장 높은 수준의 권한으로 실행" 체크
   
2. **"조건"** 탭:
   - ❌ "컴퓨터의 AC 전원이 켜져 있을 때만 작업 시작" 체크 해제
   - ❌ "작업을 실행하기 위해 대기할 때 컴퓨터 절전 모드 해제" 체크

3. **"설정"** 탭:
   - ✅ "작업이 실패할 경우 다시 시작 간격: 1분"
   - ✅ "다시 시도 횟수: 3회"

4. **확인** 클릭

## ✅ 테스트 방법

### 즉시 테스트 실행:
1. 작업 스케줄러에서 방금 만든 작업 선택
2. 오른쪽 패널에서 **"실행"** 클릭
3. 크롬이 자동으로 켜지고 포스팅이 진행되는지 확인

### 수동 실행:
프로젝트 폴더에서 `티스토리_자동_포스팅.bat` 파일을 더블클릭하면 즉시 실행됩니다.

## 🛠️ 문제 해결

### 작업이 실행되지 않는 경우:
1. Python이 시스템 PATH에 제대로 등록되어 있는지 확인
   - 명령 프롬프트에서 `python --version` 실행
2. `.env.local` 파일에 티스토리 계정 정보가 제대로 입력되어 있는지 확인
3. 작업 스케줄러 → 작업 기록 보기 → 오류 메시지 확인

### 중복 포스팅 방지:
- 스크립트가 자동으로 `posted_news_history.json` 파일을 생성하여 이미 올린 뉴스는 건너뜁니다.
- 이 파일을 삭제하면 모든 뉴스를 다시 포스팅하게 되니 주의하세요!

### 새벽 시간에 실행이 안 되는 경우:
- 컴퓨터가 절전 모드에 들어가지 않도록 설정하세요
- 전원 옵션 → 절전 모드: "사용 안 함"

---

## 📝 주의사항

1. **컴퓨터가 켜져 있어야 합니다** - 절전 모드나 종료 상태에서는 작동하지 않습니다.
2. **크롬이 자동으로 실행됩니다** - 작업 실행 시간에는 크롬을 수동으로 사용하지 마세요.
3. **중복 방지** - 같은 뉴스를 여러 번 올리지 않도록 자동으로 관리됩니다.
4. **새벽 작업** - 23:30, 03:00 작업을 위해 컴퓨터가 깨어있어야 합니다.
